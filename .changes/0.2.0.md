## 0.2.0 - 2026-06-25
### Added
* SemanticEquivalence: checks whether the AI's SQL means the same as a gold query by comparing query structure, without running either query. Reports 'equivalent' or 'unknown'.
* query_equivalence(): checks whether two queries return the same results by running them.
* FirstDecisive: combine scorers and use the first one that reaches a verdict.
### Fixed
* SemanticEquivalence: queries with redundant boolean conditions (e.g. a repeated predicate) no longer cause a slowdown when comparing query structure.
